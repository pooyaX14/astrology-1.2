Introduction

Here we present our Template - Astrology . The next generation Template which is built using HTML5, CSS3, Bootstrap3 and jQuery. The template is fully responsive and is adjustable to the screens of any size and resolution.This modern and ready to use template comes with 13 pages which includes 3 blog pages, 2 love compatibility pages, 2 signs pages, Home page and About page. It adopts a wide range of dynamic functionality taking UI to the next level.

Color Customizer

css files in the folder customizer responsible for the customization of pages (mainly primary color settings).

If you use sass, there is no need to enable css files from folder customizer. To change the primary color, you need to change the variable $general-color in the file _theme-variables.scss into the needed one.

If you don't use sass, then the main color can be changed in the appropriative file in the folder customizer.

Contact Form

This section is found in the contact_us page of this Template.

The contact form shown above comes with two services.

1. sendmail
2. smtp

By using smtp (send mail transfer protocol) as a service, we can test the functionality of the service. For example, if we want to acknowledge whether the mail sent is received by the user. once , this acknowledgement turns to be positive we can go with sendmail service.

In sendmail service the data will be sent directly to the users.

Upon Registering with mailtrap.io the username and password credentials will be generated which can be changed in config.php

The code snippet for which is shown below

                            
  1. $ user_name = '...';
  2. $ password = '...';

In the above code you can enter your own credentials.

In config.php we can change the mail address and the name to which we want to receive all the information entered by the users.

It has the following structure:

                            
  1. $ to_email = 'admin@admin.com';
  2. $ to_name = 'Admin Admin';

In the above snippet of code you can replace your own to_email and to_name credentials.

Subscribe Form

The Subscribe Form used in Astrology supports Mailchimp service. By default e-mails from the form are directly rendered to the file php/subscribe.php. once, this is done the Mailchimp gets back to the user for the comnformation. Upon, receiving the conformation from user all the data will be added to the list

we use mailchimp API key to connect with the Mailchimp Account.

The List ID is used to add mails on to the particular list of the Mailchimp and this id is unique. One needs to Register with mailchimp to get mailchimp APIKEY and mailchimp LISTID.

The code for which is shown below

                            
  1. /* Mailchimp setting */
  2. define('MC_APIKEY', ''); //Your API key from here - http://admin.mailchimp.com/account/api
  3. define('MC_LISTID', ''); //List unique id from here - http://admin.mailchimp.com/lists/
  4.  

SCSS

All SCSS files are located in the folder src/sass/. SCSS is a CSS preprocessor that adds many features like variables, mixins, imports, color manipulation, functions and tons of other powerful features.The command to run the sass file is gulp sass

we make use of laravel elixir for defining basic gulp tasks for the template.

By default, npm install will install all modules listed as dependencies in package.json .

we use bower to include all the plugins required for the project and the command to include it is bower install

we use gulp to copy only the required files from source to destination.The command for this isgulp

  • _variables.scss - imports all customized bootstrap styles
  • bootstrap.scss - file that imports all variables from all files
  • custom.scss - imports all styles
